# ☕ Exercícios de Lógica de Programação em Java

Repositório destinado ao desenvolvimento e prática de Lógica de Programação utilizando Java.

O objetivo deste projeto é consolidar os conceitos fundamentais da programação por meio da resolução de problemas, estimulando o raciocínio lógico, a capacidade de análise e a construção de algoritmos.

Durante as atividades são trabalhados conceitos essenciais como:

* Variáveis e tipos de dados
* Operadores matemáticos
* Entrada e saída de dados
* Estruturas condicionais
* Estruturas de repetição
* Vetores (Arrays)
* Manipulação de dados
* Resolução de problemas

## 🚀 Tecnologias Utilizadas

* Java
* JDK
* IDE de sua preferência (IntelliJ IDEA, Eclipse ou VS Code)

## 📚 Finalidade

Este repositório foi desenvolvido para fins de estudo e aprendizado, servindo como prática dos fundamentos da programação e da linguagem Java.

## ▶️ Como Executar

1. Clone o repositório:

```bash
git clone <url-do-repositorio>
```

2. Abra o projeto em uma IDE compatível com Java.

3. Execute o arquivo correspondente ao exercício desejado.
